# false-position-solver
False Position Method (Regula Falsi) solver for exponential equations — built with HTML, CSS, and JavaScript.
A professional web-based numerical methods solver that uses the False Position 
(Regula Falsi) algorithm with Illinois modification to find roots of exponential 
and transcendental equations. Features a live function preview, dual graphs 
(curve + convergence), iteration table, bracket evolution chart, and a 
step-by-step solution walkthrough. Built as a pure single-file HTML app — 
no dependencies, no frameworks, works on desktop and mobile.
