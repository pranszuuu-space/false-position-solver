⬡ Numerical Methods ◈ Regula Falsi ∿ Exponential Equations

False Position Method
for Exponential Equations

A professional root-finding solver using the Regula Falsi algorithm with Illinois modification — built for exponential & transcendental functions.

Method Overview
1
📌
Bracket
Choose a and b so f(a)·f(b) < 0. The Intermediate Value Theorem guarantees a root exists in [a, b].
2
📐
Interpolate
Connect (a,f(a)) and (b,f(b)) with a line. Its x-intercept estimates the root:
c = b − f(b)·(b−a)/(f(b)−f(a))
3
🔀
Update Bracket
If f(a)·f(c) < 0 → root in [a,c], set b=c. Otherwise → root in [c,b], set a=c. Repeat.
4
✓
Converge
Stop when |f(c)| < ε. Illinois modification halves stagnant endpoint values for faster convergence.
